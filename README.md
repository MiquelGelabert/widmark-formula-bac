# Widmark Formula: Blood Alcohol Content (BAC) Estimator

> **Note:** This notebook is written in **Catalan/Spanish**.

This project implements the **Widmark Formula**, a mathematical model used to estimate Blood Alcohol Content (BAC) based on various factors such as body mass, gender, amount of alcohol consumed, and time elapsed.

## About this Project 🚀

This is another of my initial projects as I learn to code. I decided to implement this formula to practice how mathematical models can be translated into Python code and how variables like gender and metabolism significantly affect the results. It's a simple but practical exercise in data visualization and modeling.

## What's Inside?

The notebook `Fórmula de Widmark.ipynb` includes:
* **Mathematical Background:** Definition of the formula and its constants ($r$ for gender distribution, $\lambda$ for elimination rate).
* **Python Implementation:** A script that calculates the BAC based on user input or predefined scenarios.
* **Comparative Analysis:** Visualizations showing how alcohol is metabolized differently over time between men and women.
* **Conclusions:** A brief reflection on the limitations of the model and its practical applications.
